# Get started

```bash
git clone https://github.com/oregor-projects/trinity-scaffolder-java.git
cd trinity-scaffolder-java
./mvnw clean install
java -jar trinity-scaffolder-java-cli/target/trinity-scaffolder-java-cli.jar 
```
